# Preguntas Frecuentes (FAQ)

## General

### ¿Qué es CodeKit Pro?
CodeKit Pro es una suite de herramientas para desarrollo asistido por IA. Gestiona prompts, genera assets, almacena snippets y documenta conocimiento técnico.

### ¿Necesito pagar?
La versión actual es gratuita. Funcionalidades premium pueden añadirse en el futuro.

### ¿Mis datos son privados?
Sí. Tus prompts, snippets y datos son privados y solo accesibles para ti (o tu equipo si trabajas en colaboración).

## Prompts

### ¿Puedo importar prompts de otros lugares?
Sí, puedes exportar/importar datos en formato JSON desde la sección de exportación.

### ¿Los prompts del sistema se pueden modificar?
Los prompts del sistema son de solo lectura, pero puedes crear copias y modificarlas.

## Herramientas

### ¿Puedo añadir mis propias herramientas?
Actualmente las herramientas están predefinidas. Esta funcionalidad puede añadirse en el futuro.

### ¿Las herramientas funcionan offline?
Algunas herramientas funcionan completamente en el navegador (formatters, generadores), otras requieren conexión.

## Técnico

### ¿Qué tecnologías usa?
- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Base de Datos: PostgreSQL
- Deploy: Docker, Docker Compose

### ¿Puedo auto-hospedarlo?
Sí, el código está disponible en GitHub y puedes desplegarlo en tu propio servidor.
